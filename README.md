# World History Simulation

## Project Overview
- **Name**: Through History - A World History Simulation
- **Goal**: Interactive 3D world history simulation built with React and Three.js
- **Features**: 
  - Interactive 3D globe visualization
  - Historical event exploration
  - Immersive educational experience

## URLs
- **Production**: https://worldhistorysimulation.pages.dev
- **Latest Deployment**: https://223b26ea.worldhistorysimulation.pages.dev

## Tech Stack
- **Frontend**: React 19.2.0
- **3D Graphics**: Three.js + React Three Fiber
- **Build Tool**: Vite 6.2.0
- **Deployment**: Cloudflare Pages
- **Icons**: Lucide React

## Data Architecture
- **Storage**: Static React application
- **Platform**: Cloudflare Pages (edge deployment)

## User Guide
This is an interactive world history simulation that allows users to explore historical events through a 3D visualization. Navigate through time and space to learn about significant moments in human history.

## Local Development
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
